# z

z: a slow, quiet, radio. enjoy.

## settings

### location

`${HOME}/.config/z`

### options

- `volume`:`float`

#### format

`name->{value};`

### example

```
volume->{0.5};
```

## development

### requirements

- [`cer0`](https://github.com/alic3dev/cer0)
- [`clic3`](https://github.com/alic3dev/clic3)
- [`interrupt_handler`](https://github.com/alic3dev/interrupt_handler)
- [`math_c`](https://github.com/alic3dev/math_c)

### build

```zsh
make
```

### clean

```zsh
make clean
```

## running

```zsh
./output/z
```

or

```zsh
make run
```
